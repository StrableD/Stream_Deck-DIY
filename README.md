# Stream_Deck-DIY

![Twitch Status](https://img.shields.io/twitch/status/Strable01?style=flat&logo=twitch&labelColor=white&color=%239146FF)
![GitHub License](https://img.shields.io/github/license/StrableD/Stream_Deck-DIY)

Ich habe mir vorgenommen ein Stream Deck mit 15 Tasten selber zu bauen.
In diesem Repository will ich meine Reise dokumentieren und damit auch eine Anleitung für andere geben.

---

**Annotation:** For the English version, check [this section](#english-version).

## Ziele

- selbstgebautes Stream Deck mit 15 Tasten
- kompatibel mit Stream Deck Software und Alternativen
- günstiger als das Original

## Aktueller Stand

Aktuell sind wir in der Planungsphase.

Für weitere Informationen zur Planung und zukünftige Umsetzung schau dir die Dateien, welche in der [folgenden Sektion](#dateien) beschrieben werden, an.
